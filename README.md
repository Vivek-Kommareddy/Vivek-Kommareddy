## 👤 About Me

I'm a **Computer Scientist** and **M.S. graduate from George Washington University (GPA: 3.9/4.0)** with hands-on experience across the full modern data and AI stack — from cloud infrastructure and software engineering to data pipelines, AI agent systems, and analytics.

I don't pick one lane. I build **end-to-end** — designing AWS architectures, shipping production APIs, orchestrating multi-agent LLM workflows, modeling data for analytics, and deploying ML systems trained on real-world datasets.

Currently working as a **Software Development Engineer** at Core Software Technologies and an **AI Fellow** at Handshake AI Solutions, pursuing opportunities where cloud, AI, data engineering, and analytics intersect at scale.

- ☁️ **Cloud** — serverless AWS architectures, event-driven pipelines, real-time ingestion, IaC
- 🛠️ **SDE** — production APIs, QR validation systems, CI/CD, Docker, REST & WebSocket services
- 🔁 **Data Engineering** — ETL/ELT pipelines, Glue, Athena, BigQuery, Firehose, dimensional modeling
- 🤖 **AI & Agents** — LLM orchestration, RAG systems, multi-agent DAGs, fine-tuning, vector stores
- 📊 **Data Analytics** — Tableau, Power BI, QuickSight dashboards, statistical modeling in R & Python

📌 **Arlington, VA** · Open to relocation · 2 Scopus-indexed papers · 1 Indian patent · Dean's Merit Award
